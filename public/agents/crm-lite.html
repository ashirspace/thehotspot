What it does
The simplest CRM you'll ever use
No menus. No modals. Click any cell, type your change, press Enter. That's it.
Inline Editing
Click any cell in the table to edit it directly. Tab to move to the next field, Enter to save. No buttons, no popups.
Instant Search
Type in the search bar to filter contacts across all fields — name, company, email, category, country — in real time.
One-Click Delete
Remove any contact with a single click on the delete button. A confirmation prompt prevents accidental deletes.
All Contacts, All Columns
See name, email, company, website, category, country, and notes — the full contact record — in a single row.
