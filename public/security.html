Security is an ongoing effort. We continuously improve controls and welcome responsible disclosure.

1. Our security approach

We use layered controls to protect user data, including scoped access, infrastructure controls, and operational review.

2. Data protection

We limit access to production data and treat campaign, contact, and Google-connected data as sensitive.

3. Access controls

Authentication, revocation, and account-level controls are designed to keep users in control.

4. Incident response

If a security incident affects users, we will investigate, contain, and communicate with appropriate urgency.
